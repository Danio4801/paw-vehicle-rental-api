# paw-vehicle-rental-api
REST API dla systemu wypożyczalni hulajnóg elektrycznych i rowerów, zbudowane przy użyciu języka Go oraz Firebase.

---

## Technologia

- **Go** 1.21
- **Firebase** (Firestore, Authentication)
- **Google App Engine**
- **JWT** do uwierzytelniania

---

## Struktura projektu 
vehicle-rental-api:
- cmd/api    
- handlers 
- models     
- services   
- middleware
- config     
- scripts    
- tests     
- docs      

### Wymagania 

- Go 1.21+
- Firebase
- GCloud SDK
